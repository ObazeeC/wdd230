Example 02: Specifying table cell relations

1 - Define relations between the header and data cells th and td.
2 - This is accomplished by introducing the scope attribute on the th elements and setting the values to the corresponding col or row value.
3 -The CSS and visual result are unchanged - the adatation provides valuabel contextual information for assistive technologies such as screen readers to help identify which cells the headers relate to.
4 - Note: If the table structure is even more complex, the (additional) use of the headers attribute on the th and td elements may improve accessibility and help assistive technologies identify th
5 - Column and row spanning: the header section now have has two rows
6 - Table caption and column summary, use the caption element as a first child of the table element
7 - Add a table footer using the tfoot element below the table body
